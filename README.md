# 📚 DAW L2 — Web Development Study Guide & Exam Prep

> Interactive HTML study guide for **L2 DAW students** at **Constantine 2 University (NTIC Faculty)**.  
> Covers all front-end topics from the Second Term Exam: HTML, CSS, JavaScript & Responsive Design.

---

## 📋 Description

A single-file interactive study guide built to help DAW L2 students prepare for the second-term web development exam. It covers every concept tested in the 2024 and 2025 exams, with beginner-friendly explanations, real code examples, and 3 full practice exams that simulate the teacher's exact questioning style — including the penalty scoring system.

---

## ✨ Features

- 🌙 **Light / Dark mode** toggle
- 🏷️ **HTML chapter** — structure, semantic elements, block vs inline, HTTP methods
- 🎨 **CSS chapter** — selectors, specificity, box model, flexbox (interactive demo!), grid, positioning
- ⚡ **JavaScript chapter** — variables, DOM manipulation, events, arrays, loops, functions
- 📱 **Responsive Design chapter** — viewport meta tag, media queries, breakpoints, touch design
- 📝 **3 full practice exams** (45–50 min each, 15 questions each)
  - Per-question **Check Answer** with color feedback (green / red / orange)
  - Full **explanations** for every answer
  - **Countdown timer** that auto-submits
  - **Score card** with grade feedback
  - **Reset** button to retry
- ⚠️ Every exam trap and fake tag from the real exams is covered

## 🎯 Exam Format Simulated

Each practice exam follows the real exam rules:

- Each question: **0.5 points**
- Select 1 wrong answer: **−0.25 points**
- Select 2+ wrong answers: **0 points**
- May have **1 to 5 correct choices** per question
